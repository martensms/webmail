Installation

Requirements

  • An IMAP or POP3 server
  • A web server (e.g. apache 1 and 2 work fine)
  • A working PHP-Module
  • Installed imap functions (e.g. php5-imap library) on the server

PHP 4 or higher is required, but PHP 4.2.0+ is recommended. PHP must be compiled with at least the --with-imap command (whether using IMAP, NNTP or POP3 support).

In addition, the following flags are needed for optional features:

--with-imap-ssl for IMAP/POP3 over SSL
--with-db for Sendmail virtusertable support
